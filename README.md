# PallaviYadav_Assesment_Incubyte
# Assessment Objective
The objective of this assessment is to validate the Gmail Compose functionality by designing comprehensive manual test cases to verify that a user can successfully send an email with the following details:                                                                                                  
1:- Subject: Incubyte                                                                                                                                      
2:- Body: QA test for Incubyte                                                                                                                       
The assessment also covers the Gmail login process, ensuring users can access the application before composing an email.
# Scope
The testing scope includes the following functionalities:                                                                                                  
1.Gmail Login                                                                                                                                                 
2.Gmail Inbox                                                                                                                                                     
3.Compose Email                                                                                                                                                   
4.Enter Recipient                                                                                                                                               
5.Enter Subject                                                                                                                                                  
6.Enter Email Body                                                                                                                                                
7.Send Email                                                                                                                                              
8.Success Notification                                                                                                                                            
9.Sent Folder Verification                                                                                                                                         
10.Draft Functionality                                                                                                                              
11.File Attachment                                                                                                                                          
12.CC and BCC                                                                                                                                                    
13.Text Formatting                                                                                                                                          
14.Hyperlink Insertion                                                                                                                                          
15.Emoji Insertion                                                                                                                                          
16.Undo Send                                                                                                                                              
Both positive and negative test scenarios are included.
# Test Approach
The assessment follows a Manual Testing approach using both Traditional Test Cases and BDD (Behavior-Driven Development) Test Cases.                       
The test cases are designed based on functional requirements and cover:                                                                             
1.Positive test scenarios                                                                                                                            
2.Negative test scenarios                                                                                                                              
3.Input validation                                                                                                                                     
4.Functional testing                                                                                                                                 
5.UI verification                                                                                                                                    
6.Error handling                                                                                                                                   
7.End-to-end email composition workflow
